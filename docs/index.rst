Getting help
============

If you've scoured the :ref:`prose <usage-docs>` and :ref:`API <api_docs>`
documentation and still can't find an answer to your question, below are
various support resources that should help. We do request that you do at least
skim the documentation before posting tickets or mailing list questions,
however!


Mailing list
------------

The best way to get help with using Schematics is via the
`schematics-dev mailing list <http://groups.google.com/group/schematics-dev>`_.
The Schematics developers do their best to reply quickly and fix reported
issues.


Bugs & Requests
---------------

Found a bug?  Want to request a feature?  Please visit Schematics's `Github Issues
<https://github.com/j2labs/schematics/issues>`_ page.
